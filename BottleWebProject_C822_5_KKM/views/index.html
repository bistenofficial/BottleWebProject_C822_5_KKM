Monte-Carlo Method

The Monte Carlo method is a statistical modeling or simulation technique. It is a numerical method for solving problems by simulating random variables. Any probabilistic problem can be solved using the Monte Carlo method, but it is justified when the drawing procedure is simpler, and not more complicated than analytical calculation.
